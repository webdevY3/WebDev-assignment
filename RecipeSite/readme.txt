Ciaran O'Connor WebDev assignment

Step 1. 	Install Xampp (v3.2.2)
Step 2. 	Place the "RecipeSite" folder in the "C:\xampp\htdocs" folder of your machine
Step 3. 	Start Apache and MySql services on xampp
Step 4. 	Visit http://localhost/phpmyadmin and add a new Database called "recipedb"
Step 5.		Click IMPORT on the top panel and select the included file "recipedb.sql", then click GO
Step 6. 	Visit http://localhost/recipeSite/Search.php to begin browsing our site
